# 🌟 Portfolio Personal - Ian Lucas

Un portafolio web moderno y responsive con navegación directa a proyectos externos.

## ✨ Características

- **Diseño Moderno**: Interfaz limpia con animaciones suaves
- **Totalmente Responsive**: Optimizado para todos los dispositivos
- **Enlaces Directos**: Los proyectos redirigen directamente a sus sitios web o repositorios
- **Navegación Intuitiva**: Click directo en proyectos para acceso inmediato
- **Filtros Dinámicos**: Sistema de filtrado por categorías
- **Animaciones Fluidas**: Efectos visuales profesionales
- **SEO Optimizado**: Estructura optimizada para motores de búsqueda

## 🚀 Estructura del Proyecto

```
portfolio/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos principales
├── js/
│   └── script.js       # Funcionalidad JavaScript
├── assets/             # Imágenes y recursos
├── unused/             # Páginas de proyecto no utilizadas
└── README.md          # Documentación
```

## 🚀 Instalación y Uso

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/IanMarcovich/mi-portfolio.git
   cd mi-portfolio
   ```

2. **Abre el proyecto**:
   - Simplemente abre `index.html` en tu navegador
   - O usa un servidor local como Live Server en VS Code

3. **Personalización**:
   - Edita el contenido en los archivos HTML
   - Modifica los estilos en `css/style.css`
   - Ajusta la funcionalidad en `js/script.js`
